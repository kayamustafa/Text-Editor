# Functionalities
- Theme selection
- Opening a new file 
- Saving the file
- Aligning text
- Finding specified text
- Replacing specified text with another one

# Design Patterns
Design patterns used in the project are:
- Abstract Factory Pattern (for theme selection)
- Singleton Pattern (to create only one scene and text area)
- Command Pattern (for all menubar operations)
